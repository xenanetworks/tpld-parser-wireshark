## Install the XMP Parser
1. Go to `Wireshark Main Window > Help > About Wireshark > Folders > Personal Lua Plugins`, click the blue Location to open plugin folder.
2. Put ``xena_tpld.lua`` into the folder.
3. Restart Wireshark
